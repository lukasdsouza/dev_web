Division

A tag <div> é usada para criar uma seção de conteúdo na página. Ela pode ser usada para criar uma estrutura lógica na página, organizar o layout ou separar conteúdo em blocos.

Existem vários tipos de listas em HTML, como <ul> (lista não ordenada), <ol> (lista ordenada) e <menu> (menu). Cada item da lista é definido pela tag <li>.
